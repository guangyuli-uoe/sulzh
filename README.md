=============  
conll dataset  
=============  
每个词占一行，  
每行的第2列为当前词语，  
第4列为当前词的词性，  
第7列为当前词的中心词的序号，  
第8列为当前词语与中心词的依存关系。  
句子与句子之间以空行间隔
